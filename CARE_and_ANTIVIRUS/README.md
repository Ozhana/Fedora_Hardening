# 🛡️ Fedora 44 Kale Matrix
## Enterprise System Maintenance & Security Suite (V8.2)

| ENGLISH | TÜRKÇE |
| :--- | :--- |
| **Introduction**<br>Kale Matrix is a zero-dependency, enterprise-grade system maintenance and security auditing suite built specifically for Fedora Linux 44. It performs comprehensive system updates, deep cleaning, disk maintenance (TRIM & BTRFS scrub), security auditing (Lynis & AIDE), and service integrity checks. The suite features live terminal monitoring with color-coded output, progress animations, and detailed before/after system snapshots. Antivirus scanning (ClamAV + Rkhunter) is provided as a separate, independent module for flexible scheduling.<br><br>**Key Features:**<br>• Live terminal monitoring with spinner animations<br>• Pre/post system snapshots for forensic comparison<br>• Smart reboot detection (kernel/glibc/systemd updates)<br>• Critical service health checks (SSH, firewall, time sync)<br>• Mount point integrity verification<br>• Automatic infected file removal (antivirus module)<br>• 30-day automatic log rotation | **Giriş**<br>Kale Matrix, Fedora Linux 44 için özel olarak geliştirilmiş, bağımlılıksız, kurumsal düzeyde bir sistem bakım ve güvenlik denetim paketidir. Kapsamlı sistem güncellemeleri, derin temizlik, disk bakımı (TRIM ve BTRFS scrub), güvenlik denetimi (Lynis ve AIDE) ve servis bütünlük kontrolleri gerçekleştirir. Paket, renk kodlu çıktı, ilerleme animasyonları ve detaylı öncesi/sonrası sistem durum kayıtları ile canlı terminal izleme özelliği sunar. Antivirüs taraması (ClamAV + Rkhunter), esnek zamanlama için ayrı ve bağımsız bir modül olarak sağlanır.<br><br>**Temel Özellikler:**<br>• Dönen animasyonlu canlı terminal izleme<br>• Adli karşılaştırma için öncesi/sonrası sistem kayıtları<br>• Akıllı yeniden başlatma tespiti (kernel/glibc/systemd güncellemeleri)<br>• Kritik servis sağlık kontrolleri (SSH, güvenlik duvarı, zaman senkronizasyonu)<br>• Bağlantı noktası bütünlük doğrulaması<br>• Otomatik virüslü dosya temizliği (antivirüs modülü)<br>• 30 günlük otomatik log rotasyonu |
| **📁 Project Structure**<br><br>The suite consists of two independent scripts:<br><br>• `kale-bakim.sh` - Weekly system maintenance (no antivirus)<br>• `kale-antivirus.sh` - Monthly deep virus & rootkit scan<br><br>This modular design allows flexible scheduling and prevents long-running antivirus scans from delaying critical system updates. | **📁 Proje Yapısı**<br><br>Paket iki bağımsız betikten oluşur:<br><br>• `kale-bakim.sh` - Haftalık sistem bakımı (antivirüssüz)<br>• `kale-antivirus.sh` - Aylık derin virüs ve rootkit taraması<br><br>Bu modüler tasarım esnek zamanlama sağlar ve uzun süren antivirüs taramalarının kritik sistem güncellemelerini geciktirmesini önler. |
| **📋 Requirements**<br><br>• Fedora Linux 44 (or newer)<br>• Root (sudo) access<br>• Internet connection (for updates)<br><br>**Optional but recommended:**<br>• Lynis: `sudo dnf install lynis -y`<br>• AIDE: `sudo dnf install aide -y`<br>• ClamAV: `sudo dnf install clamav clamav-update -y`<br>• Rkhunter: `sudo dnf install rkhunter -y`<br><br>The scripts automatically detect missing tools and continue without them. | **📋 Gereksinimler**<br><br>• Fedora Linux 44 (veya daha yenisi)<br>• Root (sudo) erişimi<br>• İnternet bağlantısı (güncellemeler için)<br><br>**İsteğe bağlı ancak önerilir:**<br>• Lynis: `sudo dnf install lynis -y`<br>• AIDE: `sudo dnf install aide -y`<br>• ClamAV: `sudo dnf install clamav clamav-update -y`<br>• Rkhunter: `sudo dnf install rkhunter -y`<br><br>Betikler eksik araçları otomatik tespit eder ve onlarsız devam eder. |
| **Step 1: Install Recommended Tools**<br>Install the security auditing and antivirus packages for full functionality.<br><br>```bash<br>sudo dnf install lynis aide clamav clamav-update rkhunter -y<br>``` | **Adım 1: Önerilen Araçları Yükleyin**<br>Tam işlevsellik için güvenlik denetimi ve antivirüs paketlerini yükleyin.<br><br>```bash<br>sudo dnf install lynis aide clamav clamav-update rkhunter -y<br>``` |
| **Step 2: Download the Scripts**<br>Clone or download both scripts to your system.<br><br>```bash<br>cd /usr/local/bin/<br>sudo curl -O https://raw.githubusercontent.com/Ozhana/Fedora_Kale_Matrix/main/kale-bakim.sh<br>sudo curl -O https://raw.githubusercontent.com/Ozhana/Fedora_Kale_Matrix/main/kale-antivirus.sh<br>``` | **Adım 2: Betikleri İndirin**<br>Her iki betiği sisteminize klonlayın veya indirin.<br><br>```bash<br>cd /usr/local/bin/<br>sudo curl -O https://raw.githubusercontent.com/Ozhana/Fedora_Kale_Matrix/main/kale-bakim.sh<br>sudo curl -O https://raw.githubusercontent.com/Ozhana/Fedora_Kale_Matrix/main/kale-antivirus.sh<br>``` |
| **Step 3: Set Execute Permissions**<br>Make both scripts executable by root only.<br><br>```bash<br>sudo chown root:root /usr/local/bin/kale-bakim.sh<br>sudo chown root:root /usr/local/bin/kale-antivirus.sh<br>sudo chmod 700 /usr/local/bin/kale-bakim.sh<br>sudo chmod 700 /usr/local/bin/kale-antivirus.sh<br>``` | **Adım 3: Çalıştırma İzinlerini Ayarlayın**<br>Her iki betiği yalnızca root tarafından çalıştırılabilir yapın.<br><br>```bash<br>sudo chown root:root /usr/local/bin/kale-bakim.sh<br>sudo chown root:root /usr/local/bin/kale-antivirus.sh<br>sudo chmod 700 /usr/local/bin/kale-bakim.sh<br>sudo chmod 700 /usr/local/bin/kale-antivirus.sh<br>``` |
| **Step 4: Run System Maintenance**<br>Execute the main maintenance script for weekly system care.<br><br>```bash<br>sudo /usr/local/bin/kale-bakim.sh<br>```<br><br>The script will:<br>• Take a system snapshot before starting<br>• Check critical services (SSH, firewall, time sync)<br>• Verify mount points<br>• Update all packages<br>• Clean orphan packages and cache<br>• Run disk TRIM and BTRFS scrub<br>• Perform security audit (Lynis + AIDE)<br>• Harden file permissions<br>• Take an after snapshot for comparison<br>• Offer system shutdown after completion | **Adım 4: Sistem Bakımını Çalıştırın**<br>Haftalık sistem bakımı için ana bakım betiğini çalıştırın.<br><br>```bash<br>sudo /usr/local/bin/kale-bakim.sh<br>```<br><br>Betik sırasıyla şunları yapacaktır:<br>• Başlamadan önce sistem durum kaydı alır<br>• Kritik servisleri kontrol eder (SSH, güvenlik duvarı, zaman)<br>• Bağlantı noktalarını doğrular<br>• Tüm paketleri günceller<br>• Yetim paketleri ve önbelleği temizler<br>• Disk TRIM ve BTRFS scrub çalıştırır<br>• Güvenlik denetimi yapar (Lynis + AIDE)<br>• Dosya izinlerini sertleştirir<br>• Karşılaştırma için son durum kaydı alır<br>• Tamamlanınca sistem kapatma seçeneği sunar |
| **Step 5: Run Antivirus Scan (Monthly)**<br>Execute the standalone antivirus module for deep malware scanning.<br><br>```bash<br>sudo /usr/local/bin/kale-antivirus.sh<br>```<br><br>The script will:<br>• Update ClamAV virus definitions (live output)<br>• Update Rkhunter database (live output)<br>• Perform rootkit scan with real-time findings<br>• Perform deep virus scan on /home, /tmp, /var/tmp<br>• Automatically remove infected files<br>• Generate detailed scan reports | **Adım 5: Antivirüs Taraması Çalıştırın (Aylık)**<br>Derin zararlı yazılım taraması için bağımsız antivirüs modülünü çalıştırın.<br><br>```bash<br>sudo /usr/local/bin/kale-antivirus.sh<br>```<br><br>Betik sırasıyla şunları yapacaktır:<br>• ClamAV virüs tanımlarını günceller (canlı çıktı)<br>• Rkhunter veritabanını günceller (canlı çıktı)<br>• Gerçek zamanlı bulgularla rootkit taraması yapar<br>• /home, /tmp, /var/tmp üzerinde derin virüs taraması yapar<br>• Virüslü dosyaları otomatik siler<br>• Detaylı tarama raporları oluşturur |
| **Step 6: Schedule with Cron (Optional)**<br>Automate weekly maintenance and monthly antivirus scans.<br><br>```bash<br>sudo crontab -e<br>```<br><br>Add these lines:<br><br>```bash<br># Weekly system maintenance - Every Sunday at 03:00<br>0 3 * * 0 /usr/local/bin/kale-bakim.sh<br><br># Monthly antivirus scan - 1st of every month at 04:00<br>0 4 1 * * /usr/local/bin/kale-antivirus.sh<br>``` | **Adım 6: Cron ile Zamanlayın (İsteğe Bağlı)**<br>Haftalık bakım ve aylık antivirüs taramalarını otomatikleştirin.<br><br>```bash<br>sudo crontab -e<br>```<br><br>Şu satırları ekleyin:<br><br>```bash<br># Haftalık sistem bakımı - Her Pazar 03:00<br>0 3 * * 0 /usr/local/bin/kale-bakim.sh<br><br># Aylık antivirüs taraması - Her ayın 1'i 04:00<br>0 4 1 * * /usr/local/bin/kale-antivirus.sh<br>``` |
| **Step 7: View Logs**<br>All logs are stored in the user's desktop log directory.<br><br>```bash<br># View maintenance logs<br>ls -la ~/Desktop/oto_gorevler/logfiles/bakim_*.log<br><br># View antivirus logs<br>ls -la ~/Desktop/oto_gorevler/logfiles/antivirus_*.log<br><br># View system snapshots (before/after comparison)<br>ls -la ~/Desktop/oto_gorevler/logfiles/snapshots/<br>``` | **Adım 7: Logları Görüntüleyin**<br>Tüm loglar kullanıcının masaüstü log dizininde saklanır.<br><br>```bash<br># Bakım loglarını görüntüleyin<br>ls -la ~/Desktop/oto_gorevler/logfiles/bakim_*.log<br><br># Antivirüs loglarını görüntüleyin<br>ls -la ~/Desktop/oto_gorevler/logfiles/antivirus_*.log<br><br># Sistem durum kayıtlarını görüntüleyin (öncesi/sonrası)<br>ls -la ~/Desktop/oto_gorevler/logfiles/snapshots/<br>``` |

Live freshclam and rkhunter update output<br>• **Rootkit Scan Progress** - Real-time file checking status<br>• **Virus Scan Progress** - Currently scanning file displayed<br>• **Infection Alerts** - Red alerts with virus names and file paths<br>• **Summary Report** - Total scanned, infected, and removed files | **🎯 Çalışma Sırasında Görecekleriniz**<br><br>Bakım betiği canlı terminal çıktısı sağlar:<br>• **ASCII Art Banner** - Başlangıçta Kale Matrix logosu<br>• **Sistem Durum Kaydı** - Başlamadan önce disk, RAM, CPU durumu<br>• **Servis Sağlığı** - SSH, güvenlik duvarı vb. için yeşil/kırmızı durum<br>• **Bağlantı Kontrolü** - /, /home, /boot bağlantılarının doğrulaması<br>• **Güncelleme İlerlemesi** - Gerçek zamanlı paket indirme durumu<br>• **Disk Bakımı** - TRIM ve BTRFS scrub canlı çıktısı<br>• **Lynis Skoru** - Renk kodlu güvenlik sıkılaştırma skoru (0-100)<br>• **Akıllı Yeniden Başlatma Uyarısı** - | **⚠️ IMPORTANT NOTES**<br><br>1. Both scripts require root privileges and will refuse to run otherwise<br>2. The maintenance script offers system shutdown after completion - press `N` to cancel<br>3. The antivirus script **automatically deletes** infected files (`--remove=yes`)<br>4. A lock file prevents running multiple instances simultaneously<br>5. Systemd-inhibit prevents sleep/suspend during execution<br>6. Logs older than 30 days are automatically cleaned<br>7. Missing optional tools (Lynis, AIDE, ClamAV, Rkhunter) are gracefully skipped | **⚠️ ÖNEMLİ NOTLAR**<br><br>1. Her iki betik de root yetkisi gerektirir, aksi halde çalışmayı reddeder<br>2. Bakım betiği tamamlanınca sistem kapatma seçeneği sunar - iptal için `N` tuşu<br>3. Antivirüs betiği virüslü dosyaları **otomatik siler** (`--remove=yes`)<br>4. Kilit dosyası aynı anda birden fazla örneğin çalışmasını engeller<br>5. Systemd-inhibit çalışma sırasında uyku/askıya almayı önler<br>6. 30 günden eski loglar otomatik temizlenir<br>7. Eksik isteğe bağlı araçlar (Lynis, AIDE, ClamAV, Rkhunter) sorunsuz atlanır |
| **🔧 Customization**<br><br>To change the log directory, edit the `LOG_DIR` variable in both scripts:<br><br>```bash<br>LOG_DIR="/your/custom/path/logfiles"<br>```<br><br>To change scan targets in the antivirus script, edit the `clamscan` command:<br><br>```bash<br>clamscan -r /home /your/custom/path --remove=yes<br>```<br><br>To disable automatic shutdown, remove or comment out the countdown section at the end of `kale-bakim.sh`. | **🔧 Özelleştirme**<br><br>Log dizinini değiştirmek için her iki betikteki `LOG_DIR` değişkenini düzenleyin:<br><br>```bash<br>LOG_DIR="/sizin/ozel/yolunuz/logfiles"<br>```<br><br>Antivirüs betiğinde tarama hedeflerini değiştirmek için `clamscan` komutunu düzenleyin:<br><br>```bash<br>clamscan -r /home /sizin/ozel/yolunuz --remove=yes<br>```<br><br>Otomatik kapanmayı devre dışı bırakmak için `kale-bakim.sh` sonundaki geri sayım bölümünü kaldırın veya yorum satırı yapın. |
