# Aegis USB Telemetry Radar

Zero-Trust USB Diagnostic and Intelligence Engine

| ENGLISH | TÜRKÇE |
| :--- | :--- |
| **1. Architecture & Philosophy**<br><br>The `usb-radar` is not just a script; it is a **State-Aware** telemetry engine designed for enterprise environments. It safely monitors kernel modules (`usb-storage`, `uas`), the USBGuard authorization matrix, and hardware transport layers without generating system entropy. It is deliberately decoupled from execution scripts (`usb-kapat` / `usb-ac`) to adhere to micro-segmentation principles. | **1. Mimari ve Felsefe**<br><br>`usb-radar` sıradan bir betik değil, kurumsal ortamlar için tasarlanmış **Durum Farkındalıklı** (State-Aware) bir telemetri motorudur. Sistem entropisi yaratmadan çekirdek modüllerini (`usb-storage`, `uas`), USBGuard yetkilendirme matrisini ve donanım taşıma katmanlarını güvenle izler. Mikro-segmentasyon prensipleri gereği, yürütme betiklerinden (`usb-kapat` / `usb-ac`) kasıtlı olarak izole edilmiştir. |
| **2. Key Features**<br><br>• **Fault-Tolerant:** Implements Graceful Degradation; survives service crashes and outputs safe diagnostic logs.<br>• **Deterministic Parsing:** Uses strict `awk` regex instead of fragile text scraping (e.g., handles exact `TRAN` and `TYPE` validations for block devices).<br>• **Automation Ready:** Native `--json` output designed for n8n or Python dashboard integrations.<br>• **Safe Monitoring:** POSIX-compliant `trap` handling for the `--watch` mode prevents terminal corruption during real-time monitoring. | **2. Temel Özellikler**<br><br>• **Hata Toleransı:** Zarif Çöküş (Graceful Degradation) uygular; servis hatalarında ayakta kalır ve güvenli teşhis logları üretir.<br>• **Deterministik Ayrıştırma:** Kırılgan metin kazıma yerine katı `awk` regex kullanır (Blok cihazlar için kesin `TRAN` ve `TYPE` doğrulaması yapar).<br>• **Otomasyona Hazır:** n8n veya Python panelleri için yerleşik `--json` çıktısı sunar.<br>• **Güvenli İzleme:** `--watch` modu için POSIX uyumlu `trap` yönetimi, canlı izleme sırasında terminalin bozulmasını engeller. |

Installation Protocol**<br><br>Download the telemetry engine and seal it in the secure binary path (`/usr/local/bin`).<br><br>**Run the following commands:**<br>`sudo curl -o /usr/local/bin/usb-radar https://raw.githubusercontent.com/Ozhana/Fedora_Hardening/main/USB-Telemetry/usb-radar`<br><br>`sudo chmod 700 /usr/local/bin/usb-radar` | **3. Kurulum Protokolü**<br><br>Telemetri motorunu indirin ve güvenli sistem dizinine (`/usr/local/bin`) mühürleyin.<br><br>**Aşağıdaki komutları çalıştırın:**<br>`sudo curl -o /usr/local/bin/usb-radar https://raw.githubusercontent.com/Ozhana/Fedora_Hardening/main/USB-Telemetry/usb-radar`<br><br>`sudo chmod 700 /usr/local/bin/usb-radar` |
| **4. Usage & Capabilities**<br><br>The radar reads system states without altering them (Read-Only).<br><br>• **Standard Matrix:**<br>`sudo usb-radar`<br>*(Outputs a 4-dimensional security state evaluation)*<br><br>• **API / Automation Mode:**<br>`sudo usb-radar --json`<br>*(Outputs raw JSON format for API consumption)*<br><br>• **Real-Time Overwatch:**<br>`sudo usb-radar --watch`<br>*(Continuous monitoring of the USB hardware tree)* | **4. Kullanım ve Yetenekler**<br><br>Radar, sistem durumunu hiçbir değişiklik yapmadan salt-okunur (Read-Only) olarak tarar.<br><br>• **Standart Matris:**<br>`sudo usb-radar`<br>*(4 boyutlu güvenlik durumu değerlendirmesi sunar)*<br><br>• **API / Otomasyon Modu:**<br>`sudo usb-radar --json`<br>*(Otomasyon sistemleri için saf JSON çıktısı verir)*<br><br>• **Gerçek Zamanlı Gözlem:**<br>`sudo usb-radar --watch`<br>*(USB donanım ağacının kesintisiz canlı izlemesini yapar)* |
