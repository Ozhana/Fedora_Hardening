| **Introduction**<br>This repository houses **Aegis Firewall V7.2**, a high-security, enterprise-grade network hardening framework optimized for Fedora. It shifts the system into a strict `DROP` target environment, deploys a zero-trust network perimeter, and enforces a strict tamper-proof `--lockdown-on` mechanism. Crucially, it features a transactional rollback subsystem to handle configuration errors autonomously and is fully compatible with internal Docker bridges (protecting production n8n automation suites from blind drops). | **Giriş**<br>Bu depo, Fedora için optimize edilmiş, yüksek güvenlikli ve kurumsal düzeyde bir ağ sıkılaştırma çerçevesi olan **Aegis Firewall V7.2** sürümünü barındırır. Sistemi katı bir `DROP` (Paketleri sessizce yut) hedef ortamına taşır, sıfır-güven bir ağ çeperi örer ve kurcalamaya karşı dayanıklı katı bir `--lockdown-on` mekanizması uygular. En kritik özellik olarak, yapılandırma hatalarını otonom olarak yöneten işlemsel geri yükleme (rollback) altyapısına sahiptir ve dahili Docker köprüleriyle (n8n otomasyon suite'lerini kör çökmelerden koruyarak) %100 uyumludur. |
