# Fapolicyd Safe Authorizer
## Zero-Trust Application Execution & Telemetry Framework

| ENGLISH  | TÜRKÇE |
| :--- | :--- |
| **1. What is this project?**<br><br>This framework deploys **fapolicyd** (File Access Policy Daemon) on your system. It acts as an ultimate shield, monitoring every application trying to run. Currently, it is set to **Permissive (Observation) Mode** for 14 days. This means it will not block your daily apps (like Docker, Xournal++, or browsers) but will quietly log what *would* be blocked.<br><br>This is a "Zero-Trust" and "Zero-Entropy" deployment. It takes full backups before touching anything and automatically reverts if an error occurs. | **1. Bu proje nedir?**<br><br>Bu altyapı, sisteminize **fapolicyd** (Dosya Erişim Politikası Hizmeti) kurar. Çalışmaya çalışan her uygulamayı denetleyen nihai bir kalkandır. Şu an için 14 günlük **Gözlem (Permissive) Modunda** çalışacak şekilde ayarlanmıştır. Yani günlük uygulamalarınızı (Docker, Xournal++ vb.) engellemez, ancak neyi engelleyeceğini sessizce kaydeder.<br><br>Bu kurulum "Sıfır Güven" ve "Sıfır Entropi" prensibiyle çalışır. Sisteme dokunmadan önce tam yedek alır ve hata olursa otomatik olarak geri döner. |
| **2. Prerequisites**<br><br>- **OS:** Fedora Linux (BTRFS recommended).<br>- **Privileges:** Root (Administrator) access.<br>- **Knowledge Level:** Absolute Beginner (Just follow the steps carefully). | **2. Ön Koşullar**<br><br>- **İşletim Sistemi:** Fedora Linux (BTRFS disk yapısı önerilir).<br>- **Yetki:** Root (Yönetici) ayrıcalıkları.<br>- **Bilgi Seviyesi:** Tam Acemi (Sadece adımları dikkatlice takip edin). |
| **3. Installation Step 1: Core Shield**<br><br>We will create a specific folder in your Home directory to keep things clean and know exactly where our files are.<br><br>**A. Open Terminal:** Press the `Super` (Windows) key, type "Terminal" and hit Enter.<br><br>**B. Create a Workspace:** Type these commands one by one and hit Enter:<br>`cd ~`<br>`mkdir aegis-installer`<br>`cd aegis-installer`<br>*(You are now in the `/home/yourusername/aegis-installer` folder)*<br><br>**C. Create the Script File:**<br>`nano aegis-fapolicyd.sh`<br><br>**D. Paste the Code:** Copy the V8-LTS script code from the source and paste it into the terminal (Right-click -> Paste).<br><br>**E. Save and Exit:** Press `Ctrl+O`, hit `Enter`, then press `Ctrl+X`.<br><br>**F. Make Executable & Run:**<br>`chmod +x aegis-fapolicyd.sh`<br>`sudo ./aegis-fapolicyd.sh`<br>*(Type your user password if asked. It won't show characters while typing, this is normal).* | **3. Kurulum Adım 1: Ana Kalkan**<br><br>Dosyaların kaybolmaması ve tam olarak nerede çalıştığımızı bilmek için "Ev" (Home) dizininizde özel bir klasör oluşturacağız.<br><br>**A. Terminali Açın:** `Super` (Windows) tuşuna basın, "Terminal" yazın ve Enter'a basın.<br><br>**B. Çalışma Alanı Oluşturun:** Şu komutları sırayla yazıp Enter'a basın:<br>`cd ~`<br>`mkdir aegis-installer`<br>`cd aegis-installer`<br>*(Şu an `/home/kullaniciadiniz/aegis-installer` klasörünün içindesiniz)*<br><br>**C. Betik Dosyasını Oluşturun:**<br>`nano aegis-fapolicyd.sh`<br><br>**D. Kodu Yapıştırın:** V8-LTS betik kodunu kopyalayıp terminale yapıştırın (Sağ tık -> Yapıştır).<br><br>**E. Kaydedip Çıkın:** Klavyenizden `Ctrl+O` tuşlarına basın, `Enter`'a basın, ardından `Ctrl+X` ile çıkın.<br><br>**F. Çalıştırılabilir Yapın ve Başlatın:**<br>`chmod +x aegis-fapolicyd.sh`<br>`sudo ./aegis-fapolicyd.sh`<br>*(Sorulursa şifrenizi girin. Yazarken karakterler görünmez, bu normaldir).* |
| **4. Installation Step 2: Telemetry Radar**<br><br>Now we need to set up the nightly reporter. Stay in the same terminal (`aegis-installer` folder).<br><br>**A. Create the File:**<br>`nano aegis-telemetry-init.sh`<br><br>**B. Paste the Code:** Copy the Telemetry script code and paste it.<br><br>**C. Save and Exit:** `Ctrl+O`, `Enter`, then `Ctrl+X`.<br><br>**D. Make Executable & Run:**<br>`chmod +x aegis-telemetry-init.sh`<br>`sudo ./aegis-telemetry-init.sh` | **4. Kurulum Adım 2: Telemetri Radarı**<br><br>Şimdi gece raporlayıcısını kuracağız. Aynı terminalde (`aegis-installer` klasöründe) kalın.<br><br>**A. Dosyayı Oluşturun:**<br>`nano aegis-telemetry-init.sh`<br><br>**B. Kodu Yapıştırın:** Telemetri betik kodunu kopyalayıp yapıştırın.<br><br>**C. Kaydedip Çıkın:** `Ctrl+O`, `Enter`, ardından `Ctrl+X`.<br><br>**D. Çalıştırılabilir Yapın ve Başlatın:**<br>`chmod +x aegis-telemetry-init.sh`<br>`sudo ./aegis-telemetry-init.sh` |
| **5. How to Read the Daily Reports**<br><br>The system will now quietly monitor your apps. Every night at exactly 23:59, it will generate a clean report of any apps that triggered the security rules.<br><br>**To view the daily report, type:**<br>`cat /var/log/aegis_telemetry.report`<br><br>**What you need to do:**<br>For the next 7 to 14 days, check this report daily. It will show us which safe apps need to be "trusted" before we fully lock down the system into Enforcing mode. | **5. Günlük Raporlar Nasıl Okunur?**<br><br>Sistem artık uygulamalarınızı sessizce izleyecek. Her gece tam 23:59'da güvenlik kurallarını tetikleyen uygulamaların temiz bir özetini çıkaracaktır.<br><br>**Günlük raporu görmek için şunu yazın:**<br>`cat /var/log/aegis_telemetry.report`<br><br>**Ne yapmanız gerekiyor?**<br>Önümüzdeki 7 ila 14 gün boyunca bu raporu günlük olarak kontrol edin. Sistemi tamamen kilitlemeden (Enforcing) önce hangi güvenli uygulamalara "güvenmemiz" gerektiğini bize gösterecektir. |
| **6. Emergency Rollback (Safety Net)**<br><br>Are you worried about breaking your computer? Don't be.<br>The script has a built-in "Fail-Fast & Rollback" trap. If any step fails during the installation, the script will automatically stop, undo every change it made, and restore your system configuration to the exact moment before you ran it. You don't have to fix anything manually. | **6. Acil Durum Geri Dönüşü (Güvenlik Ağı)**<br><br>Bilgisayarınızı bozmaktan mı korkuyorsunuz? Korkmayın.<br>Betik yerleşik bir "Fail-Fast & Rollback" (Anında Çök ve Geri Al) tuzağına sahiptir. Kurulum sırasında herhangi bir adım başarısız olursa, betik otomatik olarak durur, yaptığı her değişikliği geri alır ve sistem yapılandırmanızı çalıştırmadan önceki ana tam olarak geri döndürür. Manuel olarak hiçbir şeyi düzeltmenize gerek yoktur. |
