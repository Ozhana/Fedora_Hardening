# 🛡️ Aegis Network Hardening: TCP BBR & FQ Optimization

Zero-Entropy Kernel Network Stack Reconfiguration Framework

| ENGLISH | TÜRKÇE |
| :--- | :--- |
| **Overview**<br>Standard Linux TCP algorithms (Cubic/Reno) halve their speed upon packet loss, causing "Bufferbloat" and latency. This script permanently seals Google's TCP BBR and `fq` queueing discipline into your kernel. It optimizes Wi-Fi stability, reduces battery drain, and maximizes throughput for devices like the Surface Pro 9. | **Genel Bakış**<br>Standart Linux TCP algoritmaları (Cubic/Reno) paket kaybı anında hızı yarıya düşürerek "Bufferbloat" (tampon şişmesi) ve gecikmeye sebep olur. Bu betik, Google'ın TCP BBR ve `fq` kuyruk disiplinini çekirdeğinize kalıcı olarak mühürler. Surface Pro 9 gibi cihazlarda Wi-Fi kararlılığını artırır, batarya tüketimini azaltır ve veri akışını maksimize eder. |
| **Enterprise-Grade Architecture**<br>- **Idempotency:** Zero disk I/O if already configured.<br>- **Atomic Rollback:** Complete restoration of RAM and Disk states if any step fails.<br>- **POSIX Integrity:** Uses `sync -f` and atomic `mv` to prevent 0-byte corruption during power loss.<br>- **Fail-Fast:** Strictly validates kernel modules via Sysfs before making persistent changes. | **Kurumsal Düzey Mimari**<br>- **Sıfır Entropi:** Sistem zaten ayarlıysa disk I/O işlemi yapmaz.<br>- **Atomik Geri Alma:** Herhangi bir hata anında RAM ve Disk durumunu %100 eski haline döndürür.<br>- **POSIX Bütünlüğü:** Güç kaybında 0-bayt yozlaşmasını önlemek için `sync -f` ve atomik `mv` kullanır.<br>- **Hızlı Çöküş (Fail-Fast):** Kalıcı ayar yazmadan önce Kernel modüllerini Sysfs üzerinden doğrular. |

| **Usage (Event-Driven)**<br>Execute the tool. It uses "Cognitive Friction" to prevent accidents. You must type **PERMANENT** to apply changes.<br><br>`sudo aegis-net-bbr` | **Kullanım (Olay Güdümlü)**<br>Aracı çalıştırın. Kazaları önlemek için "Kognitif Sürtünme" kullanır.
